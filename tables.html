TABLES

INTRODUCTION


A webpage can be made more informative and understandable using tables.

The HTML tables allow web authors to arrange data like text, images, links, other tables, etc. into rows and columns of cells.

TABLES FORMATION

Table heading can be defined using < th > tag. This tag will be put to replace < td > tag, which is used to represent actual data cell. Normally you will put your top row as table heading as shown below, otherwise you can use < th > element in any row. Headings, which are defined in < th > tag are centered and bold by default.

USE THE FOLLOWING

  • Use the HTML < table > element to define a table
  • Use the HTML < tr > element to define a table row
  • Use the HTML < td > element to define a table data
  • Use the HTML < th > element to define a table heading
  • Use the HTML < caption > element to define a table caption
  • Use the CSS border property to define a border
  • Use the CSS border-collapse property to collapse cell borders
  • Use the CSS padding property to add padding to cells
  • Use the CSS text-align property to align cell text
  • Use the CSS border-spacing property to set the spacing between cells
  • Use the colspan attribute to make a cell span many columns
  • Use the rowspan attribute to make a cell span many rows
  • Use the id attribute to uniquely define one table

    1 Adding border to a table

    To add border to a table we have to use css

    		 table, th, td
    { border: 1px solid black; }
    2 Border-Collapse

    If you want the border to collapse into one solid border this property is used

    		table, th, td
    { border: 1px solid black;
    border-collapse: collapse; }
    3 Padding

    This property is used to specify the space between cell content cell border

    		 th, td
    { padding: 15px; }
    4 Text-align

    This property is used to align the headings of the table.they can be aligned to centre,left and right.

    		 th
    { text-align: left; }
    5 Border-spacing

    This property is used to specift the space between the cells.

    		table
    { border-spacing: 5px; }
    6 Colspan

    This property is used to make a cell span more than one column

    		< th colspan="2" > cell content < /th >
    7 Rowspan

    This property is used to make a cell span more than one row

    		 < th rowspan="2" > /th>
    8 Caption

    This property is used to add a caption to the table

    		 < caption > Monthly savings </caption >
    9 Special style

    This property is used to add special style to the table. The attribute used is id.

    		 table#t01
    { width: 100%; background-color: #f1f1c1; }
